# Prediccion-Satisfaccion-Clientes-en-Hoteles-y-Turismo
Proyecto de análisis y predicción de satisfacción de clientes en el sector hotelero y turístico mediante técnicas de Machine Learning y NLP. Incluye procesamiento de reseñas de TripAdvisor, Booking y Google Maps usando SQL, PySpark, TF-IDF, BERT y LSTM, con métricas de evaluación como Accuracy, F1-score y Matriz de Confusión.
